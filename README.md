# BGIS EDA

An exploratory data analysis project on BGIS data, focused on understanding patterns, distributions, and relationships across the dataset using Python.

![Animation](assets/bgis_animation.gif)

## Project Overview
This project performs exploratory data analysis on the BGIS dataset to identify key trends, data quality issues, and actionable insights. The analysis includes data cleaning, feature exploration, statistical summaries, and visual storytelling through plots and charts.

## Objectives
- Inspect the structure and quality of the dataset.
- Clean missing, inconsistent, and duplicate values.
- Explore numerical and categorical features.
- Identify trends, correlations, and outliers.
- Present insights using clear visualizations.
